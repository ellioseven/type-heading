# Type Heading - BETA

A responsive typography tool.

Type heading is currently in **BETA**. It should only be used for testing and/or
development, not production work.

## Requirements

To use Type Heading, you will need at least:

- Ruby: 2.1.5
- Sass 3.3.13
- Compass 1.0.1
